url.js is a 420 byte library for getting variables out of your url.

Basic Usage
url Expression Result
host/?foo=bar url.foo "bar"
host/?debug=true url.bool("debug") true
host/?speed=5 url.num("speed") 5
Defaults
host/ url.bool("debug", true) true
host/?debug=false url.bool("debug", true) false
host/ url.num("speed", 10) 10
More
host/?debug url.bool("debug") true
host/#about url.hash "about"
