url.js is a tiny library for getting variables out of your url.

get source view on github
Basic Usage
host.com/?foo=bar url.foo "bar"
host.com/?debug=true url.bool("debug") true
host.com/?speed=5 url.num("speed") 5
Defaults
host.com/ url.bool("debug", true) true
host.com/?debug=false url.bool("debug", true) false
host.com/ url.num("speed", 10) 10
More
host.com/?debug url.bool("debug") true
host.com/#about url.hash "about"
